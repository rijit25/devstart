Bienvenue sur **DevStart**, votre hub d'entraînement intensif pour devenir un expert du développement Web Fullstack. Cette application est conçue comme un terrain de jeu interactif ("sandbox") pour pratiquer, tester et valider vos connaissances via des défis de code et des quiz.

## 🚀 Fonctionnalités Principales

- **Tableau de bord centralisé** : Suivez votre progression globale sur toutes les technologies.
- **Modules d'Entraînement** :
  - **HTML Expert** : Architecture sémantique, accessibilité & structure.
  - **CSS Master** : Flexbox, Grid, Animations & Design System.
  - **JavaScript** : DOM, Événements, API & Asynchrone.
  - **PHP Modern** : POO, PDO, Sessions & Sécurité.
  - **MySQL** : Requêtes complexes & Modélisation.
  - **Architecture & Sécurité (Nouveau 🛡️)** : MVC, XSS, CSRF & Best Practices.
- **Défi 1000 Labs** : Un générateur infini d'exercices procéduraux pour chaque technologie. Atteignez les 1000 labs pour devenir un maître !
- **Mode Expert** : Validation stricte du code (syntaxe, quotes, standards).
- **Gamification** : Système de points, barres de progression et badges de niveau (Junior, Intermédiaire, Production Ready).
- **Design Moderne** : Interface "Glassmorphism" soignée et responsive.

## 🛠️ Installation & Utilisation

Ce projet est une application web statique (HTML/CSS/JS) qui peut être lancée directement dans n'importe quel navigateur moderne. 
Pour la partie PHP/MySQL, un serveur local est nécessaire (WAMP, XAMPP, Laragon ou Docker).

### Prérequis

- Un navigateur web moderne (Chrome, Firefox, Edge).
- Un environnement PHP local (recommandé pour les parties PHP/MySQL) :
  - [XAMPP](https://www.apachefriends.org/fr/index.html)
  - [Laragon](https://laragon.org/) (Recommandé sur Windows)
  - [MAMP](https://www.mamp.info/) (Mac)

### Lancement

1.  Clonez ce dépôt :
    ```bash
    git clone https://github.com/votre-username/devstart.git
    ```
2.  Accédez au dossier du projet :
    ```bash
    cd devstart
    ```
3.  Ouvrez le fichier `index.html` dans votre navigateur.
    - *Pour les exercices PHP/SQL* : Placez le dossier dans le répertoire `www` ou `htdocs` de votre serveur local et accédez-y via `http://localhost/devstart`.
