🏗️ Architecture & Clean Code

Ne codez plus comme un amateur. Structurez vos projets pour qu'ils soient maintenables.

Le Pattern MVC (Model - View - Controller)

Séparez la logique (Controller), les données (Model) et l'affichage (View). Ne mélangez jamais de requêtes SQL dans vos fichiers HTML !

Structure de Dossiers Standard
  • /public : Seul dossier accessible au web (index.php, css, js)
  • /src : Classes et logique métier (Controllers, Models)
  • /templates : Vues (Fichiers .php ou .html.twig)
  • /config : Configuration (BDD, clés API)
Routing

Au lieu de créer `contact.php`, `about.php`, utilisez un point d'entrée unique (`index.php`) qui redirige vers le bon contrôleur en fonction de l'URL.

🔒 Sécurité Web Critique

La sécurité n'est pas une option. Protégez vos utilisateurs.

XSS (Cross Site Scripting)

L'attaquant injecte du JavaScript malveillant.
Solution : Échapper TOUTES les sorties avec htmlspecialchars().

Injection SQL

L'attaquant manipule vos requêtes BDD.
Solution : Utiliser TOUJOURS des requêtes préparées (PDO).

CSRF (Cross-Site Request Forgery)

L'attaquant force l'utilisateur à faire une action à son insu.
Solution : Tokens CSRF dans les formulaires.
