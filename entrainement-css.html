🎯 Le Sniper CSS (Les Sélecteurs)

La "science" du CSS commence ici. Comment cibler précisément les éléments HTML sans toucher aux autres ?

/* HTML: <div id="header"> <a class="btn">...</a> </div> */ /* 1. Cibler l'élément avec l'ID "header" */ { ... } /* 2. Cibler TOUS les éléments avec la classe "btn" */ { ... } /* 3. Cibler les liens (<a>) qui sont DANS une liste (<ul>) */ ul { ... } /* 4. Cibler uniquement le survol de la souris sur un bouton */ .btn { ... } /* 5. Cibler TOUS les éléments de la page (Universel) */ { margin: 0; } /* 6. Cibler le paragraphe immédiatement après un titre h1 (Adjacent) */ h1 p { ... }

📐 L'Architecture Moderne (Flexbox)

Pour centrer des éléments ou créer des colonnes, Flexbox est incontournable. Complétez ce code pour centrer parfaitement un élément.

.container { /* 1. Activer le mode Flexbox */ display: ; /* 2. Centrer horizontalement (Axe principal) */ justify-content: ; /* 3. Centrer verticalement (Axe secondaire) */ align-items: ; /* 4. Mettre les enfants en colonne (au lieu de ligne) */ flex-direction: ; /* 5. Espacer les éléments (l'écart entre eux) */ gap: ; }

📦 Design System & Box Model

Reproduisez le style d'une "Carte Produit" moderne (Arrondis, ombres, espacement).

.card { /* 1. Espace INTÉRIEUR (entre le texte et le bord) */ : 20px; /* 2. Espace EXTÉRIEUR (entre la carte et les autres) */ : 15px; /* 3. Arrondir les coins de 10px */ border-: 10px; /* 4. Ombre portée (Esthétique) */ box-: 0 4px 6px rgba(0,0,0,0.1); /* 5. Changer le curseur en main au survol */ cursor: ; /* 6. Utiliser une couleur transparente (Alpha) */ background-color: rgba(255, 255, 255, ); }

▦ La Grille CSS (Grid Layout)

Grid est plus puissant que Flexbox pour les mises en page 2D (Lignes + Colonnes). Créez une galerie de 3 colonnes.

.gallery { /* 1. Activer le mode Grid */ display: ; /* 2. Définir 3 colonnes égales (1 fraction chacune) */ grid-template-columns: 1fr 1fr; /* 3. Espace de 15px entre les cellules */ gap: ; } .featured-item { /* 4. Faire en sorte que cet élément prenne 2 colonnes de large */ grid-column: span ; }

📱 Mobile First & Variables

Utilisez des variables pour le thème et une Media Query pour adapter le site aux téléphones.

:root { /* 1. Déclarer une variable globale pour la couleur */ --main-color: #ff0000; } .button { /* 2. Utiliser la variable définie plus haut */ background: (--main-color); } /* 3. Règle pour les écrans de taille MOBILE (max 600px) */ @media (: 600px) { .container { /* 4. Passer les éléments en colonne sur mobile */ flex-direction: ; } }
